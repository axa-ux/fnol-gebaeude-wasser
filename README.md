# BYB – Schadenmeldung (FNOL) · Prototyp

Klickbarer, konversationeller Prototyp einer **Gebäude-Wasserschaden-Schadenmeldung** für Usertesting.

## Öffnen
`index.html` im Browser öffnen – eigenständige Datei, kein Build, kein Backend. Desktop-optimiert (~1280 px).

## Inhalt
- Skriptbasierter Happy-Path (Persona „Corinna Naville"), „controlled smart" UX
- Freitext-Auto-Extraktion, Chips statt Dropdowns, NLU für Datum/Betrag
- Live-Zusammenfassung „Das weiss ich bisher", editierbare Antworten (✎)
- Tippindikator/Streaming, Deckungsprüfung, Abschluss mit Schadennummer

> Hinweis: Die Marke „BYB" ist fiktiv (kein AXA-Bezug). Schriften via Google Fonts mit System-Fallback.
